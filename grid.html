CSS Grid

CSS Grid is a two-dimensional layout system — the most powerful tool for building complex, responsive page layouts.

Items can span multiple columns or rows with grid-column: span N.

3. Named Grid Areas

The most readable way to define complex layouts — name each region and assign items to them.

Header
Sidebar
Main Content
Footer
.layout {
  display: grid;
  grid-template-areas:
    'header header'
    'sidebar main'
    'footer footer';
  grid-template-columns: 200px 1fr;
}
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }
