Exercise 02

CSS Selectors

Master the art of targeting HTML elements precisely using the full power of CSS selectors.

1. Class & ID Selectors

Use . for classes (reusable) and # for IDs (unique).

.highlight — targeted by class
.active — targeted by another class
.highlight { background: rgba(167,139,250,0.2); }
#unique-id  { color: cyan; }

2. Element & Type Selectors

Target all elements of a specific HTML tag type.

This <p> with class "special" is styled via: p.special

p.special { color: #fb7185; font-style: italic; }

3. Pseudo-class Selectors

Style elements based on their state or position.

  • First item — styled with li:first-child
  • Second item — styled with li:nth-child(even)
  • Third item — normal
  • Last item — styled with li:last-child
li:first-child  { color: #a78bfa; }
li:last-child   { color: #38bdf8; }
li:nth-child(even) { background: rgba(167,139,250,0.06); }

4. Attribute Selectors

Target elements based on their HTML attributes.

input[type="text"]  { border-color: #a78bfa; }
input[type="email"] { border-color: #38bdf8; }

5. Combinators

Combine selectors to express relationships between elements.

CombinatorExampleMeaning
A Bdiv pAll <p> inside <div>
A > Bdiv > pDirect child <p> of <div>
A + Bh2 + pFirst <p> immediately after <h2>
A ~ Bh2 ~ pAll <p> siblings after <h2>
