Exercise 08

JavaScript DOM

The Document Object Model lets JavaScript read and dynamically manipulate every element, attribute, and style on the page.

1. Selecting Elements

document.getElementById('myId')
document.querySelector('.myClass')
document.querySelectorAll('p')

2. Changing Content & Style

Click a button to change me!

document.getElementById('change-target').textContent = 'New text!';
element.style.color = '#38bdf8';
element.style.fontSize = '1.5rem';

3. Creating & Appending Elements

Items will appear here...

const el = document.createElement('div');
el.textContent = 'New item';
el.className   = 'tag';
container.appendChild(el);

4. Modifying Classes

Toggle my classes 
element.classList.add('active');
element.classList.remove('active');
element.classList.toggle('active');

5. Reading & Setting Attributes

element.getAttribute('placeholder')
element.setAttribute('placeholder', 'New text')
element.value  // for form inputs
