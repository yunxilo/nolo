JavaScript Events

Events are signals from the browser that something happened — a click, a keystroke, a hover. JavaScript listens and responds.

1. Mouse Events

Hover or click me!
element.addEventListener('click',      handler);
element.addEventListener('mouseover',  handler);
element.addEventListener('mouseout',   handler);
element.addEventListener('dblclick',   handler);

2. Keyboard Events

input.addEventListener('keydown', e => console.log(e.key));
input.addEventListener('keyup',   e => console.log(e.key));
input.addEventListener('input',   e => console.log(e.target.value));

3. Form Events

4. Event Delegation

Instead of adding listeners to every child, add one to the parent and check event.target.

parent.addEventListener('click', e => {
  if (e.target.matches('button')) {
    console.log('Clicked:', e.target.dataset.action);
  }
});
