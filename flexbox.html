Flexbox Layout

Flexbox is a one-dimensional layout method for arranging items in rows or columns with powerful alignment controls.

1. flex-direction

Controls whether items flow in a row (default) or column.

Row (default):

Item 1
Item 2
Item 3

Column:

Item 1
Item 2
Item 3
.container { display: flex; flex-direction: row | column; }

2. justify-content

Aligns items along the main axis (horizontal for rows).

space-between:

A
B
C

center:

A
B
C

space-evenly:

A
B
C
.container { justify-content: flex-start | center | space-between | space-evenly; }

3. align-items

Aligns items along the cross axis (vertical for rows).

Short
Tall
Medium
.container { align-items: stretch | center | flex-start | flex-end; }

4. flex-grow and flex-wrap

flex-grow lets items expand to fill available space. flex-wrap allows items to wrap to the next line.

Grows
Grows
Full Width Row
.item { flex-grow: 1; }   /* share remaining space equally */
.container { flex-wrap: wrap; } /* items wrap on overflow */

5. A Centered Card Layout

Centering content is trivially easy with flexbox.

Perfectly Centered
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 150px;
}
