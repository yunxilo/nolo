Navigation & Links

Navigation is the backbone of usability. HTML links combined with smart CSS patterns create every kind of navigation pattern.

1. Anchor Links and Targets

The <a> tag creates hyperlinks. The href attribute specifies the destination.

Internal Page (/) External (GitHub) ↗ Anchor (#section) Mailto Link 
<a href="page.html">Internal link</a>
<a href="https://site.com" target="_blank" rel="noopener">External</a>
<a href="#section-id">Anchor link</a>
<a href="mailto:user@example.com">Email</a>

2. Breadcrumb Navigation

Breadcrumbs show the user's current location within a site hierarchy.

Home / Web Programming / Navigation Exercise 
<nav aria-label="Breadcrumb">
  <a href="/">Home</a> /
  <a href="/exercises">Exercises</a> /
  <span>Current Page</span>
</nav>

3. Tab Navigation

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page semantically.

CSS (Cascading Style Sheets) is the language used to style HTML documents. It describes how elements should be displayed on screen, paper, or in other media.

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It enables interactive and dynamic web content.

5. Skip Navigation (Accessibility)

A "skip to main content" link lets keyboard users bypass the navigation and jump directly to the content.

<a href="#main-content" class="skip-link">Skip to main content</a>
<nav>...navigation...</nav>
<main id="main-content">...content...</main>
