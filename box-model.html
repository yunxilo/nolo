CSS Box Model

Every HTML element is a rectangular box. The CSS box model describes how margin, border, padding, and content stack together.

Box Model Visualized

MARGIN (30px)
BORDER (8px)
PADDING (20px)
CONTENT

1. Content

The innermost layer. This is where text, images, and child elements live. You control its size with width and height.

.box { width: 200px; height: 100px; }

2. Padding

Padding is transparent space between the content and the border. It adds breathing room inside the element.

padding: 0.5rem (small)
padding: 1.5rem (medium)
padding: 3rem (large)
.box { padding: 1rem; }           /* all sides */
.box { padding: 1rem 2rem; }      /* top/bottom left/right */
.box { padding: 10px 15px 20px 5px; } /* top right bottom left */

3. Border

The border wraps around the padding and content. You can control its width, style, and color.

solid
dashed
dotted
double
.box { border: 2px solid #a78bfa; }
.box { border-radius: 0.5rem; } /* rounded corners */

4. Margin

Margin is transparent space outside the border. It creates separation between elements.

.box { margin: 1rem; }          /* all sides */
.box { margin: 0 auto; }        /* center horizontally */
.box { margin-top: 2rem; }      /* one side only */

5. box-sizing: border-box

By default, padding and border are added on top of the declared width. With box-sizing: border-box, width includes padding and border — making layouts predictable.

*, *::before, *::after {
  box-sizing: border-box; /* Best practice — applied globally */
}
