Responsive Design

Responsive design means your page looks great on any screen — phone, tablet, laptop, or widescreen monitor.

1. Current Breakpoint

Resize the browser window to see which breakpoint is active:

📱 XS — Mobile (<480px) 📱 SM — Small (481–768px) 💻 MD — Tablet (769–1024px) 🖥️ LG — Desktop (>1025px) 
@media (max-width: 480px)  { /* mobile  */ }
@media (max-width: 768px)  { /* tablet  */ }
@media (max-width: 1024px) { /* laptop  */ }
@media (min-width: 1025px) { /* desktop */ }

2. Fluid Typography with clamp()

clamp(min, preferred, max) scales the font size fluidly between a minimum and maximum value.

This text scales with the viewport
h1 { font-size: clamp(1.5rem, 5vw, 3.5rem); }

4. The meta viewport Tag

Without this tag, mobile browsers render the page at desktop width and scale it down. It must be in every page's <head>.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

5. Mobile-First Strategy

Write base styles for mobile, then use min-width media queries to enhance for larger screens.

/* Base = mobile */
.nav { flex-direction: column; }

/* Tablet and up */
@media (min-width: 768px) { .nav { flex-direction: row; } }

/* Desktop and up */
@media (min-width: 1024px) { .nav { gap: 3rem; } }
