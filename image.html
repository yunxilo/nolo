Exercise 01

Images in HTML

Learn how to embed, size, describe, and group images using HTML5 semantic elements.

What are HTML Images?

The <img> tag embeds images into a web page. It is a void element — it has no closing tag. The src attribute specifies the image source (URL or file path) and the alt attribute provides a text alternative for screen readers and when the image fails to load.

1. Basic Image with Alt Text

Always include an alt attribute. This is essential for accessibility and SEO.

<img src="photo.jpg" alt="A sample landscape photo">

2. Controlling Image Size

You can set the width and height with HTML attributes or CSS. Always specify both to prevent layout shift.

<img src="image.jpg" alt="Description" width="200" height="150">

3. Figure and Figcaption

Use <figure> and <figcaption> to semantically associate a caption with an image.

<figure>
  <img src="landscape.jpg" alt="A beautiful landscape">
  <figcaption>Fig. 1 — A beautiful landscape.</figcaption>
</figure>

4. Responsive Images

Use CSS max-width: 100% so images never overflow their container on small screens.

img { max-width: 100%; height: auto; }

5. Image as a Link

Wrap an image in an <a> tag to make it clickable.

<a href="page.html">
  <img src="thumbnail.jpg" alt="Click to navigate">
</a>

Key Takeaways

  • ✓ Always use the alt attribute for accessibility
  • ✓ Specify width and height to prevent layout shift
  • ✓ Use <figure> and <figcaption> for semantic captions
  • ✓ Apply max-width: 100% for responsiveness
