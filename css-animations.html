CSS Animations

CSS animations bring interfaces to life without a single line of JavaScript — using keyframes, transitions, and hardware-accelerated transforms.

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-20px); }
}
.element { animation: bounce 1s ease-in-out infinite; }

2. CSS Transitions

Transitions smoothly interpolate property changes triggered by state changes like :hover.

Scale
Glow
Color
.card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}
.card:hover {
  transform: translateY(-4px) scale(1.02);
  box-shadow: 0 20px 40px rgba(0,0,0,0.3);
}

3. Animated Progress Bar

@keyframes growBar {
  from { transform: scaleX(0); }
  to   { transform: scaleX(1); }
}
.fill { animation: growBar 2s ease forwards; transform-origin: left; }

4. Shimmer / Skeleton Loading

A shimmer effect simulates loading content — pure CSS, no JavaScript.

.shimmer::after {
  content: '';
  background: linear-gradient(90deg, transparent, rgba(255,255,255,0.1), transparent);
  animation: slide 1.5s ease infinite;
}
