Course Summary

Through these 12 exercises I developed a solid, practical foundation in front-end web development. Starting from the very basics of HTML document structure and semantic markup, I progressed through visual design with CSS (box model, flexbox, grid, animations), and finally added interactivity with JavaScript DOM manipulation and event handling. Each exercise built on the previous one, resulting in a complete understanding of how modern, responsive, accessible websites are built from the ground up.
